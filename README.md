NameDrop Monorepo

NameDrop is a monorepo that contains a Fastify backend API and an Expo (React Native) mobile app.

Tech Stack
- Fastify (API)
- Node.js (API runtime)
- Expo (Mobile)
- React Native (Mobile)
- TypeScript (Mobile)
- pnpm Workspaces (Monorepo)
