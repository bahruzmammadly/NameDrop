# NameDrop

NameDrop is an open-source Android app for fast contact sharing using **QR codes** and **NFC**.  

## Features
- QR contact sharing
- NFC tap sharing
- Minimal UI
- Fast and lightweight
- Privacy-first

## Tech Stack

**Mobile**
- React Native
- TypeScript

**Backend**
- Node.js
- Express.js
- MongoDB
